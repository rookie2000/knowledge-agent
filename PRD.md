# 产品需求文档 (PRD)

## 项目概述

个人知识库问答 Agent，让用户可以上传文档，然后通过自然语言提问获取答案。

## 核心功能

### P0 - 必须实现

1. **文档上传与处理**
   - 支持 PDF 和 Markdown 文件
   - 自动分块（chunk）并建立向量索引
   - 存储到本地 ChromaDB

2. **智能问答**
   - 用户提问 → 检索相关文档片段 → LLM 生成回答
   - 回答需要引用来源（哪篇文档、哪个片段）
   - 支持多轮对话，记住上下文

3. **Tool Use**
   - Agent 可以调用工具：
     - `search_knowledge` - 搜索知识库
     - `get_document_info` - 获取文档元信息
   - 展示 Agent 的推理和工具调用过程

### P1 - 后续迭代

- Web UI 界面
- 更多文档格式支持（Word、网页）
- 联网搜索能力
- 对话历史持久化

## 用户场景

1. 用户上传几篇技术文档，然后问"XXX 的原理是什么？"
2. Agent 检索相关片段，给出答案并标注来源
3. 用户追问"能详细说说吗？"，Agent 基于上下文继续回答

## 验收标准

- [ ] 能成功上传 PDF/Markdown 并建立索引
- [ ] 问答能准确检索并回答基于文档内容的问题
- [ ] 回答包含来源引用
- [ ] 多轮对话上下文正确
- [ ] Agent 能自主决定是否调用工具
