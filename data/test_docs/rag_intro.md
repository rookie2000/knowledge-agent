# RAG 简介

## 什么是 RAG?

RAG (Retrieval-Augmented Generation) 是一种结合检索和生成的 AI 技术架构。它通过从外部知识库中检索相关信息，然后将这些信息作为上下文提供给大语言模型，从而生成更准确、更有依据的回答。

## RAG 的工作原理

RAG 的工作流程分为三个主要步骤：

1. **检索 (Retrieval)**：当用户提出问题时，系统首先从知识库中检索与问题相关的文档片段。

2. **增强 (Augmentation)**：将检索到的相关文档片段与用户问题一起构建成提示词。

3. **生成 (Generation)**：大语言模型基于增强后的提示词生成最终回答。

## RAG 的优势

- **减少幻觉**：基于真实文档生成回答，减少模型编造内容
- **知识可更新**：只需更新知识库，无需重新训练模型
- **可追溯性**：可以引用来源，便于验证
- **成本效益**：比微调模型更经济实惠

## RAG 的应用场景

- 企业知识库问答
- 客户服务系统
- 技术文档助手
- 法律/医疗文档分析
